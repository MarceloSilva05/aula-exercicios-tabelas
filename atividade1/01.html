TABLE A tag table define uma tabela HTML.
TH A tag th define uma célula de cabeçalho em uma tabela HTML.
TR A tag tr define uma linha em uma tabela HTML.
TD A tag td define uma célula de dados padrão em uma tabela HTML.
THEAD A tag thead é usada para agrupar o conteúdo do cabeçalho em uma tabela HTML.
TBODY A tag tbody é usada para agrupar o conteúdo do corpo em uma tabela HTML.
TFOOT A tag tfoot é usada para agrupar o conteúdo do rodapé em uma tabela HTML.
ALIGN O atributo align em HTML é usado para especificar o alinhamento do conteúdo do texto do Elemento.
BORDER As propriedades da border CSS permitem que você especifique o estilo, a largura e a cor da borda de um elemento.
CAPTION A tag caption define uma titulo da tabela.
ROWSPAN O rowspanatributo especifica o número de linhas que uma célula deve abranger.
COLSPAN O colspan atributo define o número de colunas que uma célula deve abranger.
WIDTH O width atributo especifica a largura do elemento, em pixels.
